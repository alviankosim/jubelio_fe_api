## Jubelio Fullstack Web Ecommerce
### `Getting Started`
This is a app test for Jubelio Full Stack App.

### `Installing`
to install all dependencies use this command below in each folder (fe and api)
```
npm install
```
or
```
yarn install
```
### `Starting`
to start use 

API
```
cd api
```
```
node server.js
```

FE
```
cd fe
```
```
npm start
```
